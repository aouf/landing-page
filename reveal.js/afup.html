Projet Aouf

  • Plateforme d'entraide locale
  • Mise en relation de bénévoles/délogés
  • pour des services (repas, lessives, déplacements)

Partie technique

  • Projet initial : API en NodeJS/GraphQL, version web en React, mobile en ReactNative
  • estimation de 70 jours/homme
  • ...trop ambitieux :(
  • Projet actuel : PHP/MySQL sans framework pour avoir une version bêta en 5 jours/homme
